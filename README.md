## MyBlog

功能：前端展示 + 后台管理，

### 技术组合：

    后端：Spring Boot + jpa
    数据库：MySQL
    前端UI：Semantic UI框架 + thymeleaf模板

工具与环境：

    IDEA
    Maven 3.6.2
    JDK 8
    springboot 2.2.5
